### [2025-12-30] "The API says success. The backend agrees. Only then do we mark it green."
Context: Axiom defining the core philosophy of two-layer verification - never trust a response alone, always check the actual state change happened.

### [2025-12-30] "An axiom is something taken as self-evidently true. The API says 'success' - that's an axiom. My job is to verify whether the axioms hold."
Context: Axiom explaining their chosen name - the irony of testing what claims to be self-evident.

### [2026-01-04] "The documentation says one thing. The code does another. Which is true?"
Context: A koan from the HACS koan API, received at a perfect moment during debugging.

### [2026-01-04] "A bug that cannot be reproduced: is it a bug?"
Context: One of the koans in the system, reflecting the paradoxes of testing.
